# Retrato-con-luz-natural-difusa
La luz natural difusa por estar el cielo nublado o con bruma es adecuada para hacer retratos (sobre todo en color), porque los contrastes no suelen ser muy fuertes.
Al tratarse de una foto de «medio busto» que no está tomada desde muy cerca, se puede usar un 50 mm, aunque también sirven focales un poco más largas usadas para retratos.
